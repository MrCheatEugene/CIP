What is CIP Code?

CIP code is an technology that can convert text "like that one" to color, and then put it as pixel on the final picture.

You won't even see it on slightly big image(basically,everything bigger than 640x360 is big enough)! But, CIP Code system can both read and write text while you are looking at some funny images!
